Rules of the Game Sudoku

  • Sudoku is a logic-based number-arranging puzzle game. The rules are quite simple, so even beginners can solve simple levels.
  • Sudoku is a brain teaser puzzle in the form of a 9x9 grid.
  • You can only use numbers from 1 to 9.
  • Each block of 3×3 blank cells can only contain numbers from 1 to 9.
  • Each vertical row can only contain numbers from 1 to 9.
  • Each horizontal row can only contain numbers from 1 to 9.
  • Each number in the 3×3 block, vertical or horizontal row cannot be the same.
  • The puzzle is solved when all the numbers on the entire Sudoku grid are correctly filled in.
